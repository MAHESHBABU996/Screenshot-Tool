# Screenshot-Tool
# 📸 Pro Snipping Tool (Python Screenshot App)

A Python-based desktop screenshot application built using **Tkinter**.  
It allows users to take full-screen screenshots, snip selected areas, use global hotkeys, and automatically organize screenshots by date.

---

## 🚀 Features

- 📸 Full-screen screenshot capture  
- ✂️ Snip (select area) screenshot tool  
- ⌨️ Global hotkey support (Ctrl + Shift + S)  
- 📁 Automatic folder creation by date  
- 📋 Copy screenshot path to clipboard  
- 🖼️ Instant preview after capture  
- 📜 Session-based screenshot history  

---

## 🛠️ Tech Stack

- Python 3.x  
- Tkinter (GUI)  
- Pillow (Image processing)  
- keyboard (Hotkeys)  
- pyperclip (Clipboard handling)  
- threading (Background tasks)  
- datetime, os, time (built-in modules)
